# Examples

Standalone examples demonstrating hypervector concepts and APIs. Each includes a runnable Python script.

| Example | Description |
|---------|-------------|
| [Operators from Scratch](operators/index.md) | Implement bind, release, and bundle using only the low-level offset API |
| [Mexican Dollar](mexican_dollar/index.md) | Analogical reasoning: "What's the Dollar of Mexico?" |
| [Bulk Storage Benchmark](bulk_storage/index.md) | Populate and query storage backends at scale |
| [Words Index](words_index/index.md) | Encode 5,000 English words as letter-Sparkle sequences; query by exact word or by positional suffix with multi-attractor NNS |
| [LISP Interpreter](pylisp/index.md) | A full LISP interpreter where every data structure is a hypervector |
