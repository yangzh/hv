# Examples

Standalone runnable scripts under [`examples/`](https://github.com/yangzh/hv/tree/main/examples) — each demonstrates a different facet of hypervector computing. Click through for the walkthrough.

| Example | What it shows |
|---------|---------------|
| [Mexican Dollar](mexican_dollar/index.md) | Analogical reasoning of *"What's the Dollar of Mexico?"*: bind/bundle as the math behind analogy. |
| [Word Indexer](word_indexer/index.md) | Encoding and novel queries for 5,000 English words. |
| [Bulk Storage Benchmark](bulk_storage/index.md) | Populate various substrates with thousands of chunks and measure retrieval performance. |
| [Operators from Scratch](operators/index.md) | Reimplement `bind` and `bundle` in pure Python — the core math underneath the library. |
| [LISP Interpreter](pylisp/index.md) | A full LISP where every atom, cons cell, and environment is a hypervector. For the VSA-curious. |
